# QBlog2

### 介绍

基于 Vue 2 + SpringBoot 的个人博客系统

### 主要功能

博客前台
- 博客展示
- 博客分类
- 博客标签
- 时间线
- 博客留言

博客后台
- 管理员登录
- 写文章
- 修改文章
- 删除文章
- 文章统计
- 留言删除
- 留言回复

### 使用到的技术

后端：
- SpringBoot
- Undertow
- Lombok
- Hutool
- Knife4j
- Mybatis-Plus
- JJwt
- MinIO
- Guava
- 七牛云

数据库：
- MySQL
- Redis


前端：
- Vue 2
- Vue-Router
- Vuex
- ElementUI
- Axios
- Hightllight.js
- Showdown.js
- Github-markdown-css
- Vue-Element-Admin
- Js-cookie
- Echarts
- Mavon-Editor


部署：
- Docker
- Docker-Compose
